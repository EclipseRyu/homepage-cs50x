Table Stuff

So if I'm doing it right, a simple table will look like this:

Amelia Quinn Camden
123-456-7890 098-765-4321 102-938-4756

(Not real numbers, of course.)

Where the entire table is encapsulated in a <table> tag, table rows are encapsulated in a <tr> tag, and table columns, or "table data", the individual cells, are encapsulated in a <td> tag.

Unfortunately, it looks like shit, right? Surely we can do better?

List Stuff

So, you can also make lists and stuff, ordered and unordered.
If I'm doing it right, an unordered list should look like this:

  • Me
  • Myself
  • I
  • Amelia
  • Tranny dyke (AmeliaRyu)

Where the entire list is encapsulated with the <ul> tag,
and the items of the list are encapsulated with the <li> tag.
